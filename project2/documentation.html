Features & Special Efforts

I added a few extra features beyond the minimum requirements:

  • Pagination: I added "Next" and "Previous" buttons to go through multiple pages of API results.
  • Controls: I have 4 working controls (Search text, Platform filter, Sort by, and a Results Limit dropdown).
  • Score Badges: The games show Metacritic or User scores, with different colors based on how high the rating is.
  • Home Reset: Clicking the top logo clears the search and goes back to showing trending games.
